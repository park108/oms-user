# Overview
- Order Management System(약칭 OMS) 의 사용자를 관리하는 서비스입니다.
- OMS를 사용하는 사용자 정보(법인별 사용자, 관리자, 시스템 운영자) 정보를 관리합니다.
- 패스워드 관리 기능을 포함합니다.
